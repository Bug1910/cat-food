# 高品質貓罐頭品牌 - 動畫效果設計備註

## 已實現的動畫效果

### 1. 首屏視覺 (Hero Section)
- **淡入上升動畫 (Fade In Up)**
  - 標題「極致純肉體驗」由下往上緩緩淡入
  - 副標題「給毛孩最純粹的愛」延遲 0.2 秒淡入
  - 產品特點文字延遲 0.4 秒淡入
  - CTA 按鈕延遲 0.6 秒淡入
  - 產品圖片從右側淡入
  - 圓形貓咪圖片縮放進入

- **向下滾動指示器**
  - 持續上下彈跳動畫，提示用戶向下滾動

### 2. 專利與信任背書 (Trust Section)
- **專利證書脈動效果 (Pulse)**
  - 紅色專利印章以 2 秒週期持續脈動閃爍
  - 兩個證書的脈動效果略有延遲，創造視覺層次

- **數據統計卡片**
  - 四個統計數據依序縮放彈出
  - 每個卡片延遲 0.1 秒，創造波浪效果

### 3. 三大核心特點 (Core Features)
- **彈跳出現動畫 (Staggered Pop-up)**
  - 三個特點卡片依序從底部彈跳進入
  - 使用 Spring 動畫，彈跳係數 0.4
  - 每個卡片延遲 0.2 秒

- **滑鼠懸停效果**
  - 圖示旋轉 360 度
  - 卡片輕微放大 (scale: 1.05)
  - 卡片輕微搖擺 (rotate: -2° 到 2°)

- **進度條動畫**
  - 底部裝飾條從 0 寬度展開至 60%

### 4. 全口味產品矩陣 (Product Lineup)
- **水平視差效果 (Parallax Scroll)**
  - 偶數產品向右移動最多 30px
  - 奇數產品向左移動最多 30px
  - 移動量隨滾動進度變化

- **交錯佈局**
  - 偶數產品圖片在左，文字在右
  - 奇數產品圖片在右，文字在左

- **營養成分進度條**
  - 蛋白質、脂肪、纖維比例以動畫方式展開
  - 使用產品主題色彩的漸層

- **標籤動畫**
  - 功能標籤依序縮放彈出
  - 每個標籤延遲 0.1 秒

- **懸停效果**
  - 產品圖片放大 1.05 倍
  - 購物車按鈕點擊縮小至 0.95 倍

### 5. 食材與質地細節 (Macro Details)
- **浮動食材元件**
  - 綠葉圖示：上下飄動 20px，輕微旋轉
  - 水滴圖示：上下飄動 20px（反向），輕微旋轉
  - 愛心圖示：上下飄動 15px，縮放動畫
  - 所有浮動元件使用不同的動畫週期（3.5-4.5 秒）

- **視差滾動**
  - 背景元件根據滾動位置產生不同速度的移動
  - 創造空間深度感

- **特點卡片**
  - 從右側滑入
  - 懸停時陰影加深

### 6. 常見問題與訂閱 CTA (FAQ & Subscription)
- **風琴式摺疊選單**
  - 點擊問題時平滑展開/收合
  - 箭頭圖示旋轉 180 度
  - 高度和透明度同步動畫

- **呼吸燈效果 (Breathing Light)**
  - CTA 按鈕持續呼吸燈動畫
  - 透明度在 0.5 到 1 之間變化
  - 縮放在 1 到 1.05 之間變化
  - 週期 2 秒，無限重複

- **信任標記**
  - 三個優勢標記依序淡入

### 7. 導航欄 (Navigation)
- **初始進入動畫**
  - 從頂部滑入 (Y: -100 到 0)

- **滾動變化**
  - 滾動超過 50px 時背景變為白色半透明
  - 增加模糊效果和陰影

- **手機選單**
  - 展開/收合時高度和透明度平滑過渡

### 8. 其他互動元素
- **回到頂部按鈕**
  - 滾動超過 500px 時淡入並縮放進入
  - 懸停時放大 1.1 倍
  - 點擊時縮小至 0.9 倍

- **優惠彈窗**
  - 5 秒後自動彈出
  - 背景遮罩淡入
  - 彈窗使用 Spring 動畫從底部彈出
  - 禮物圖示持續搖擺和縮放

## 顏色系統
- **主色調**：柔和米色 (#F3F4F6, amber-50)
- **口味色彩**：
  - 鮪魚：海洋藍 (blue-400 to blue-600)
  - 雞肉：雞肉橙 (orange-400 to orange-600)
  - 牛肉：紅色 (red-400 to red-600)
  - 鮭魚：粉紅 (pink-400 to pink-600)
  - 鮮蝦：青綠 (teal-400 to teal-600)
  - 鴨肉：琥珀 (amber-400 to amber-600)

## 響應式設計
- 手機端：單欄佈局，簡化動畫
- 平板端：雙欄佈局，保留大部分動畫
- 桌面端：完整佈局和所有動畫效果

## 性能優化
- 使用 `once: true` 避免重複觸發動畫
- 使用 `useInView` 只在元素進入視窗時播放動畫
- 使用 CSS transform 而非 position 來提升性能
- 適當使用 `will-change` 提示瀏覽器優化
